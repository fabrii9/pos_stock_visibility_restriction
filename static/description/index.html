Key Features

Real-time Stock Visibility

Displays current stock levels for all products in the POS interface, updated dynamically as transactions occur.

Out-of-Stock Restrictions

Prevents sales of unavailable products by automatically restricting them based on current inventory levels.

Hide Unavailable Products

Option to completely hide out-of-stock products from the POS interface for a cleaner user experience.

Configuration

Configure the module through Point of Sale settings:

  • Navigate to  Point of Sale → Configuration → Settings
  • Enable  "Hide Zero Stock"  option
  • Enable  "Restrict Zero Stock"  option
  • Save configuration and restart POS sessions or just refresh the POS page
